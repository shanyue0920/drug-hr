# 培训体系与能力发展（维H6）

> 本 reference 解决"能力怎么养的体系"。**关键边界：HR 只做培训体系设计与落地管理（架构/节奏/考核闭环/能力评估/导师机制），具体培训内容（产品知识/疾病知识/合规红线/拜访技能）归后续独立的「医药培训 Skill」。HR 不写培训课件，只搭体系并确保内容由对应主责角色提供。**

## H6.1 培训体系架构

| 维度 | 设计要点 |
|---|---|
| 分层 | 新人（入职闭环）/在职（持续提升）/经理（领导力） |
| 分类 | 产品（→医药培训 Skill，医学证据→医学 skill）/合规（→医药培训 Skill 或合规 skill）/技能（→医药培训 Skill）/管理（通用方法论） |
| 频次 | 新人集中 2–4 周 + 在岗带教 1–3 月；在职年度 + 季度更新；经理项目制 |
| 形式 | 集中营/线上学院/区域会/导师带教/行动学习 |

> 内容供给责任：产品、合规、技能三类培训内容由医药培训 Skill（及医学/合规 skill）提供，HR 负责排期、组织、考核闭环。

## H6.2 新人训练营机制（转正前闭环）

- 流程：集中培训（内容由医药培训 Skill 提供）→ 在岗带教 → 转正考核。
- HR 管：考核门槛（须全部通过）、节奏（周期 2–4 周 + 带教 1–3 月）、达标判定。
- 转正门槛：培训达标 + 试用期目标达成（见 H3.5）。**培训内容（合规红线/产品知识/拜访技能）不在此写，由医药培训 Skill 提供。**

## H6.3 经理领导力培训体系

| 层级 | 重点（体系侧） |
|---|---|
| 新任地区经理 | 角色转换、目标拆解、一对一辅导、团队会议（内容可引通用管理方法论） |
| 大区经理 | 区域策略、跨地区协同、人才梯队、商业敏锐 |
| 总监 | 业务组合、组织设计、高层对齐 |

- 形式：领导力项目 + 导师（上级带教）+ 行动学习。业务内容接口对应主责 skill。

## H6.4 能力评估与 IDP（个人发展计划）

- 测评：基于 H3.1 胜任力模型，年度测评（自评+上级评）。
- IDP：针对短板定 3–5 项发展行动（培训/带教/轮岗/项目），季度复盘。
- 输出：能力九宫格（业绩×潜力）用于人才盘点。
- 注意：测评的"能力"维度（产品/合规/技能）的具体标准由医药培训 Skill 的内容体系支撑。

## H6.5 导师带教机制

- 机制：每个新人/新经理配导师，签带教协议。
- 管理：带教周期、复盘节奏（首月陪访、周复盘）、激励（带教津贴/认可，纳入晋升评估）。
- 不含具体带教内容（内容归医药培训 Skill）。

## H6.6 培训落地与效果追踪

- 指标：培训完成率、达标率、转正通过率、经理培养达成率。
- 联动：与绩效(H4 KPI)、晋升(H5)、人才盘点(H6.4)挂钩。
- 闭环：培训—考核—应用—复盘，年度优化体系。
