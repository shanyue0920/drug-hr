# 激励体系与薪酬绩效（维H4 · 含可计算奖金模板）

> 本 reference 产出**可计算的奖金/激励数值结论**，不是文字区间。所有假设参数必须披露来源与时点，敏感分析必须给。行业薪酬基准标 💭，使用前建议以最新薪酬调研复核（截至 YYYY-MM）。

## H4.1 薪酬结构（固浮比）

| 层级 | 建议固浮比 | 说明 |
|---|---|---|
| 医药代表(MR) | 60:40 ~ 50:50 | 浮动占比高，强业绩驱动 |
| 地区经理(RM) | 65:35 ~ 55:45 | 兼顾团队管理 |
| 大区经理(RGM) | 70:30 ~ 60:40 | 更重管理+战略 |

> 固浮比随产品成熟度调整：新产品（爬坡期）固薪偏高保稳定；成熟产品浮动偏高拉增量。

## H4.2 奖金方案设计（三种常用）

### 模式A：目标奖金制（最常用）
```
个人奖金 = 目标奖金基数 × 达成率系数
```
达成率系数阶梯（示例，可调整）：
| 达成率 | 系数 | 说明 |
|---|---|---|
| <80% | 0 | 不达门槛 |
| 80%–100% | 0 ~ 1.0 线性 | 按达成比例 |
| 100%–120% | 1.0 ~ 1.5 加速 | 超额加速 |
| >120% | 1.5（或封顶 2.0） | 防极端、控成本 |

### 模式B：提成制（增量导向）
```
个人奖金 = 净销量 × 基础提成率 × 增量系数
增量系数：同比增长部分提成率更高（如 +50%）
```

### 模式C：专项奖（叠加）
- 拓岗奖：新开发医院/新进院，单家固定奖。
- 新适应症/新产品线冲刺奖：期间增量加权。
- 合规奖：合规零违规 + 系数加成（正向激励合规）。

## H4.3 KPI 指标体系（权重框架）

| 指标 | 建议权重 | 说明 |
|---|---|---|
| 销量/纯销达成 | 50%–60% | 主指标 |
| 准入(进院数/双通道) | 10%–15% | 与准入协同 |
| 客户覆盖/拜访质量 | 10%–15% | 过程指标 |
| 合规行为 | 一票否决 | 违规即取消奖金 |

> 合规须"一票否决"嵌入 KPI，而非软指标——这是 HR 与合规 skill 的硬接口。

## H4.4 奖金测算模板（⭐核心，可填数）

### 参数表（可填）
| 参数 | 符号 | 取值 | 来源/假设 |
|---|---|---|---|
| 目标奖金基数（年） | B | ____ | 薪酬结构定 |
| 实际达成率 | R | ____% | 用户提供 |
| 达成率系数（按 H4.2 阶梯） | Coef | ____ | 套阶梯 |
| 增量销量 | Inc | ____ | 同比增量 |
| 增量提成率 | r_inc | ____% | 方案定 |
| 专项奖合计 | Sp | ____ | 拓岗/冲刺 |
| 合规系数 | K_comp | 1.0 / 0 | 零违规=1，违规=0 |

### 计算
```
个人年奖金 = (B × Coef + Inc × r_inc + Sp) × K_comp
```
**示例**：
- B=10万，R=110% → Coef=1.25（100–120%档线性 1.0–1.5，110%≈1.25）
- Inc=50万，r_inc=2% → 增量奖=1万
- Sp=0.5万，K_comp=1.0
- 个人年奖金 = (10×1.25 + 1 + 0.5)×1.0 = 14.0 万

### 敏感性（必须给）
| 场景 | 达成率 | Coef | 奖金 |
|---|---|---|---|
| 悲观 | 90% | 0.5 | ____ |
| 基准 | 110% | 1.25 | 14.0万 |
| 乐观 | 125% | 1.5(封顶) | ____ |

> 90% 落在 80–100% 线性档，Coef = (90-80)/(100-80) = 0.5（未达 100% 门槛，无加速）。

## H4.5 销售竞赛与专项激励

- 周期：季度赛/半年度冲刺，避免过长失效。
- 预算：竞赛奖金池 ≤ 季度浮动薪酬的 X%（标假设）。
- 设计：目标可达（跳一跳够得着）、规则透明、即时兑现、合规挂钩。
