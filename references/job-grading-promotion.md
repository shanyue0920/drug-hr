# 职级体系与晋升通道（维H5）

> 本 reference 解决"级别怎么定、怎么晋、人才怎么储备"。双通道是关键创新——避免"千军万马挤管理独木桥"。行业基准标 💭。

## H5.1 职级地图（标准销售序列）

| 职级 | 名称 | 典型年薪带宽(💭) | 核心要求 |
|---|---|---|---|
| P1 | 医药代表 | ____ | 独立覆盖、达标 |
| P2 | 高级代表 | ____ | 稳定超额、带教新人 |
| P3 | 资深代表/首席产品专家 | ____ | 顶级专业、KOL 级沟通 |
| M1 | 地区经理 | ____ | 管 6–12 人、区域达标 |
| M2 | 大区经理 | ____ | 管多地区、策略+人才梯队 |
| M3 | 销售总监/BU Head | ____ | 全线战略、商业结果 |

> 具体年薪带宽须以最新薪酬调研 🔍 核实（见 H4.6），本表仅框架。

## H5.2 双通道（管理线 + 专业线）

- **管理线（M 序列）**：代表→地区→大区→总监，带团队、负管理责任。
- **专业线（P 序列）**：代表→高级→资深/首席专家，不带人、走专业深度（顶级产品专家、KOL 级学术影响力）。
- **等价对待**：P3 资深专家薪酬可对标 M1/M2，避免"不当官就没钱没地位"。
- **横向转换**：专业线高潜可转管理线（须过管理潜质评估，见 H3.4）。

## H5.3 晋升标准与周期

| 维度 | 门槛 | 说明 |
|---|---|---|
| 业绩 | 连续 2 期达标（≥100%） | 基础门槛 |
| 能力 | 通过上一级能力测评 | 专业线测专业深度，管理线测管理力 |
| 合规 | 零违规记录 | 一票否决 |
| 潜力 | 继任评估通过（管理岗） | 高潜识别 |

- **周期**：年度主窗口 + 半年度补窗；晋升生效与薪酬调整同步。
- **透明**：标准公开、流程可申诉，避免"领导拍脑袋"。

## H5.4 人才梯队与继任计划

- **关键岗识别**：大区/地区经理为关键岗（离职影响大）。
- **继任者地图**：每个关键岗标注继任者（Ready now / Ready 1yr / Gap）。
- **高潜池**：跨区识别高潜，轮岗+专项培养。
- ** Gap 处理**：无继任者岗位须有"应急方案"（上级兼管/外部猎聘预案）。

## H5.5 关键岗位储备

- 储备机制：高潜轮岗、影子经理、专项项目历练。
- 留存：对高潜给清晰路径+适度薪酬倾斜+认可。
- 风险：关键岗过度集中单人（" bus factor=1"），须分散。
